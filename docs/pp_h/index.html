GenBitmapFBMNoise

Create a simplex or perlin noise bitmap with FBM 

EXPORT bool GenBitmapFBMNoise(
    Bitmap *b,
    int w,
    int h,
    int offsetX,
    int offsetY,
    float scale,
    float lacunarity,
    float gain,
    int octaves);
Parameters
b

Bitmap output

w

Bitmap width

h

Bitmap height

offsetX

Noise offset

offsetY

Noise offset

scale

Noise scale

lacunarity

Controls the size of the secondary fine details (2.0 is a good start)

gain

The amount to multiply the noise's amplitude by (0.5 is a good start)

octaves

Describes the level of detail of the noise (8 is a good start)

Return Value

Boolean of success/failure

Discussion

Define PP_SIMPLEX_NOISE or PP_PERLIN_NOISE to select noise algorithm (PP_SIMPLEX_NOISE is default)
