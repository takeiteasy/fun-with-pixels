ppBegin

pp initialization function 

EXPORT int ppBegin(
    int w,
    int h,
    const char *title,
    ppFlags flags);
Parameters
w

Window width

h

Window height

title

Window title

flags

Window flags (see: ppFlags)

Return Value

Returns true if window opened correctly, false if there was an error

Discussion

This function should be called at the start of the program

ppCallbacks

Assign window event callbacks 

EXPORT void ppCallbacks(
    PP_CALLBACKS void *userdata);
Parameters
KeyboardCallback

Callback for keyboard events

MouseButtonCallback

Callback for mouse button events

MouseMoveCallback

Callback for mouse movement event

MouseScrollCallback

Callback for mouse scroll event

ResizedCallback

Callback for window resize event

FocusCallback

Callback for window focus/blur events

ClosedCallback

Callback for window close event

userdata

Userdata stored by pp, passed to callback functions

Discussion

All callbacks are optional, pass NULL to skip callback. These can also be set individually, e.g. ppKeyboardCallback(cb)

ppEnd

Clean up pp 

EXPORT void ppEnd(
    void);
Discussion

This should be called at the end of the program (or to end the program) to clean up resources

ppFlush

Draw bitmap to the window 

EXPORT void ppFlush(
    int *data,
    int w,
    int h);
Parameters
data

Image data (should be formated a 32bit integer ARGB)

w

Image data width

h

Image data height

Discussion

This should be called every frame

ppPoll

Poll window events 

EXPORT int ppPoll(
    void);
Return Value

Returns true if window is still open, false if the window has closed

Discussion

This function must be called every frame. Otherwise, the window will not update

ppRunning

ppFlags

Window config flags 

typedef enum { 
    ppResizable = 1 << 0, 
    ppFullscreen = 1 << 1, 
    ppFullscreenDesktop = 1 << 2, 
    ppBorderless = 1 << 3, 
    ppAlwaysOnTop = 1 << 4 
} ppFlags;
Constants
ppResizable

Resizable window flag

ppFullscreen

Native fullscreen window flag

ppFullscreenDesktop

Fake fullscreen window flag

ppBorderless

Remove window titlebar flag

ppAlwaysOnTop

Force window to always be on top flag

Discussion

Flags can be combined with bitwise operators, e.g. ppResizable | ppAlwaysOnTop

ppMod

Keyboard modifier flags 

typedef enum { 
    KEY_MOD_SHIFT = 1 << 0, 
    KEY_MOD_CONTROL = 1 << 1, 
    KEY_MOD_ALT = 1 << 2, 
    KEY_MOD_SUPER = 1 << 3, 
    KEY_MOD_CAPS_LOCK = 1 << 4, 
    KEY_MOD_NUM_LOCK = 1 << 5 
} ppMod;
Constants
KEY_MOD_SHIFT

Shift key

KEY_MOD_CONTROL

Control key

KEY_MOD_ALT

Alt key

KEY_MOD_SUPER

'Super' key (Windows key)

KEY_MOD_CAPS_LOCK

Caps-lock key

KEY_MOD_NUM_LOCK

Num-lock key

Discussion

Modifier flags will be combined, so use bitwise operators for checks, e.g. flags & KEY_MOD_SHIFT
